# 2026-07-05 Work Log

## 淘汰赛晋级图增添比分预测

### 需求
淘汰赛晋级图（KnockoutProgress 组件）中未完成比赛只显示胜率百分比，需要增加比分预测结果。

### 实现
1. **后端 `backend/output.py`**:
   - 新增 `_poisson_pmf(k, lam)` — Poisson 概率质量函数
   - 新增 `_predict_score(team_a, team_b, feature_engine, prob_a, prob_b)` — 基于小组赛统计（场均进球/失球）+ 胜率概率，用 Poisson PMF 搜索最可能的整数比分
   - 淘汰赛衰减系数 0.72（比赛更保守）
   - 60% 统计 + 40% 概率混合
   - 修改 `_build_knockout_bracket` 接收 `feature_engine` 参数，为未完成且双方已知的比赛添加 `predScoreA`/`predScoreB`
   - 修改 `generate_predictions_json` 传递 `feature_engine` 给 `_build_knockout_bracket`

2. **前端 `frontend/src/data/loader.ts`**:
   - `KnockoutMatch` 接口添加 `predScoreA?: number` 和 `predScoreB?: number`

3. **前端 `frontend/src/components/KnockoutProgress.tsx`**:
   - `MatchCard` 组件为未完成比赛显示预测比分（蓝色 badge）+ 胜率百分比
   - 预测比分获胜方用 accent 蓝色高亮

4. **前端 `frontend/src/styles/global.css`**:
   - 新增 `.match-stats`（flex 容器）、`.pred-score`（badge 样式）、`.pred-score.win`（获胜方高亮）
