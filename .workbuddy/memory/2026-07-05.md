## 比分预测算法优化 v2

### 问题
比分预测过于保守，所有比赛预测都是 1-0 或 0-1，比分只有 0 或 1。

### 根本原因分析
- decay=0.72 衰减太狠 → xG 降至 0.6-1.4
- base=1.8 概率基础值太低
- blend=60/40 统计权重过大（统计本身已被衰减）
- 纯 Poisson MAP 在 lambda<1.5 时几乎总是选 0 或 1 球

### 修复（`backend/output.py` `_predict_score`）
| 参数 | 旧值 | 新值 | 理由 |
|------|------|------|------|
| decay | 0.72 | 0.85 | 淘汰赛场均 2.5 球 |
| prob_base | 1.8 | 3.5 | 提高预期进球基线 |
| blend (stat:prob) | 60:40 | 30:70 | 概率权重主导 |
| total_goals_floor | 无 | 2.5 | 保证最低进球期望 |
| search_range | 0-4 | 0-6 | 覆盖高分比赛 |
| draw_correction | 无 | 有 | 平局但胜率差>15%时强者+1 |
